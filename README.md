**🌌 Cosmic Clicker 🌌**

Welcome to Cosmic Clicker, a fast-paced, space-themed arcade game built with Python and Tkinter! Test your reflexes, catch falling stars, and set new high scores in this fun desktop application.


🎮 What is this game about?

In Cosmic Clicker, you are a space explorer on a mission to catch as many shooting stars as possible before they disappear into the void. The game features a personalized player profile system, allowing you to choose your own space avatar and track your personal high scores across multiple gaming sessions!


✨ Features

Player Profiles: Enter your nickname and choose from a variety of space-themed emoji avatars.
Local Database: The game automatically saves your profile and personal high score using a local JSON database (players_db.json).
Dynamic Gameplay: Stars fall at random speeds and sizes.
Rare Stars: Keep an eye out for the rare, giant Cyan stars that give double points!
Interactive UI: Custom crosshair cursors, dynamic buttons, and a celebratory alarm when you break your high score.


🕹️ How it Works (Rules & Controls)

Launch the Game: Run the Python script to open the Start Screen.
Create a Profile: Enter a nickname (English letters and numbers only, max 15 characters) and select your favorite avatar.
Start Catching: Click the Start button to begin the star shower.
Controls:
Left-Click on any star to catch it.
Yellow Stars = +1 Point.
Cyan Stars = +2 Points.
Set a Record: Catch as many stars as you can. If you beat your personal best, a special "NEW RECORD" alarm will flash on the screen!
Pause/Restart: Use the "Stop game" and "Start again" buttons to pause or reset your session at any time.


🛠️ Requirements to Run

To play this game, you need Python installed on your computer. The game uses the built-in tkinter library for the graphical interface and json for saving data, so no external downloads or pip installs are required!

Simply run: python main.py (Replace main.py with the actual name of your Python file).
